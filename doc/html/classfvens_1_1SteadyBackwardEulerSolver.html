StatusCode fvens::SteadyBackwardEulerSolver< nvars >::solve ( Vec u)

Runs the time-stepping loop with backward Euler time-stepping.

Stores timing data in a TimingData object that can be retreived by getTimingData. Throws an instance of Numerical_error if the residual becomes NaN or inf.

Parameters
[in,out]uThe solution vector containing the initial solution and which will contain the final solution on return.

Freezes the non-zero structure for efficiency in subsequent time steps.
