Detailed Description

template<typename scalar, typename j_real = freal>
class fvens::FlowBC< scalar, j_real >

Abstract class for storing the details and providing functionality for one type if BC.

Each FlowBC (derived class) object is associated with an integer tag which, in turn, is associated with a collection of boundary faces on which the BC is to be applied. Note that multiple objects of a single BC type may be required for different boundary values.

Template parameter scalar is the type used for computing the boundary value only. Type j_real is is used for computing Jacobian of the boundary state w.r.t. the interior state. By default, j_real is set the same as freal, the default type for the residual computation.
