Detailed Description

template<typename scalar, int nvars>
class fvens::L2TraceVector< scalar, nvars >

A distributed vector belonging to the trace space of a discontinuous function defined on a mesh.

Essentially a pair of "left" and "right" vectors defined on the trace of the mesh. By trace of a function over the domain, we mean its restriction to the set of all faces of the mesh. The faces are ordered in the same way as in the mesh object.

Warning
Even though this class is templated on the scalar type, it is only for future use. Currently, only the fvens::freal type works.

Connectivity face indices associated with shared faces as seen by the neighbouring subdomain.

This gives the connface index (of this subdomain) associated with each face of the other subdomain neighbouring this one. So the size is same as sharedfaces. This is necessary because the other subdomain has a different ordering of the shared connectivity faces.
