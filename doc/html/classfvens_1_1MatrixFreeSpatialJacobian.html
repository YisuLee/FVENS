Detailed Description

template<int nvars>
class fvens::MatrixFreeSpatialJacobian< nvars >

Matrix-free Jacobian of the flux.

An object of this type is associated with a specific spatial discretization context. The normalized step length epsilon for the finite-difference Jacobian is set to a default value, but it also queried from the PETSc options database.
