◆ restrictMeshToPartitions()

UMesh< freal, 2 > fvens::ReplicatedGlobalMeshPartitioner::restrictMeshToPartitions ( ) const

Computes the localized mesh on this rank given a partition of the cells of the global mesh.

  1. Copy inpoel, get local to global elem map
  1. Copy required point coords into local mesh get global to local and local-to-global point maps
  1. Convert inpoel entries from global indices to local
  1. Use point global-to-local map to identify global bfaces that are needed in this rank. Copy them. In 3D this will be N^(2/3) log n. (N is global size, n is local size)
  1. Compute local esuel and mark elements that neighbor and points that lie on a physical boundary face
  1. Use local esuel, the global esuel and local-to-global elem map to build the connectivity face structure. Use the fact that ordering of faces within elements is not changed during restriction to assign global face indices.
