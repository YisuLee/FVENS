FlowSolutionFunctionals fvens::FlowCase::run_output ( const bool surface_file_needed,
const bool vtu_output_needed,
const UMesh< freal, NDIM > & mesh,
Vec u ) const
virtual

Setup and run a case, return some functionals of interest and optionally write output files.

Whether VTU volume output and surface variable output is required to files is given by arguments here. Whether volume variable output (non-VTU) is required is taken from the options database FlowParserOptions::vol_output_reqd .

Note
Currently, output to files only works for single-process runs. However, the return values are computed in any case.
Parameters
[in]surface_file_neededTrue if the solution on relevant surfaces should be written out to files, else set to false
[in]vtu_output_neededWhether writing the full solution to VTU files is desired.
[in]meshThe mesh to solve the problem on
[in,out]uPointer to an allocated and initialized PETSc vec used for storing the solution
