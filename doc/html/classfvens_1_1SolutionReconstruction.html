Detailed Description

template<typename scalar, int nvars>
class fvens::SolutionReconstruction< scalar, nvars >

Abstract class for computing face values from cell-centered values and gradients.

Note
Face values at boundary faces are only computed for the left (interior) side. Right side values for boundary faces need to computed elsewhere using boundary conditions.
