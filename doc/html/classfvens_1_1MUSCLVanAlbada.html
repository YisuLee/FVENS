MUSCL resconstruction is compact in the cell gradients. So assuming connectivity ghost cells have updated gradients, we can compute resconstructed values without any more communication. This property is shared with linear reconstruction but is opposed to WENO reconstruction.

Implements fvens::MUSCLReconstruction< scalar, nvars >.
