Detailed Description

template<typename scalar, int nvars>
class fvens::BarthJespersenLimiter< scalar, nvars >

Non-differentiable multidimensional slope limiter for linear reconstruction.

In domain decomposition frameworks, node that this reconstruction does NOT need the cell gradients of connectivity ghost cells.
